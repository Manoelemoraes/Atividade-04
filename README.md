### Estações do ano
### *1. Introdução:*
---
**_As estações do ano acontecem por causa da inclinação da terra em relação ao sol. O movimento do nosso planeta em torno do sol, dura um ano. Esse movimento recebe o nome de translação e a sua principal conseqüência é a mudança das estações do ano._**
---
| Meses  | Estações |
---------|---------
| Janeiro | Verão🌞 |
| Fevereiro | Verão 🌞 |
| Março | Outono🍁 |
| Abril | Outono🍁 |
| Maior | Outono🍁 |
| Junho | Outono🍁 |
| Julho | Inverno❄️ |
| Agosto | Inverno❄️ |
| Setembro | Inverno❄️ |
| Outubro | Primavera🌸 |
| Novembro | Primavera🌸 |
| Dezembro | Primavera🌸 |
